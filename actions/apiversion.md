# APIVersion [API 1.4]

Returns the version of the API operating in the current responding version of ActiLife.

## Action

### Example Action
```JSON
{
    "Action": "APIVersion"
}
```

## Response

### Example Response
```JSON
{
    "Response": "APIVersion",
    "Success": true,
    "Error": "",
    "Args": { ... },
    "Payload": {
        "Version": "1.4"
    }
}
```
