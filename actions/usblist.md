# USBList [API 1.1]

Lists all connected USB devices and continues listing devices as they are plugged in.

**Notes:**

* ActiLife will send one response per device connected.
* **AntID** will only be provided for ANT capable devices.

## Action

### Example Action

    {
        "Action": "USBList"
    }
