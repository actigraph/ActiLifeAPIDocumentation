# ActiLifeVersion [API 1.6]

Returns the version of ActiLife that is running.

### Example Action
```JSON
{
    "Action": "ActiLifeVersion"
}
```

```JSON
{
    "Response": "ActiLifeVersion",
    "Success": true,
    "Error": "",
    "Args": { ... },
    "Payload": {
        "Version": "6.6"

}
```
